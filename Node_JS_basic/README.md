# Node.js Basic

This project contains basic Node.js exercises covering fundamental concepts including:

- Console output and basic JavaScript execution
- Working with process stdin
- Synchronous and asynchronous file reading
- Creating HTTP servers using Node's HTTP module
- Building web servers with Express
- Organizing a complex Express application with controllers and routes

## Files

- `0-console.js` - Display a message in console
- `1-stdin.js` - Interactive program using stdin
- `2-read_file.js` - Read file synchronously
- `3-read_file_async.js` - Read file asynchronously
- `4-http.js` - Basic HTTP server
- `5-http.js` - HTTP server with multiple routes
- `6-http_express.js` - Basic Express server
- `7-http_express.js` - Express server with multiple routes
- `full_server/` - Complete Express application with MVC structure
