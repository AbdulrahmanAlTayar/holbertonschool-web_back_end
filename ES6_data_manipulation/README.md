# ES6 Data Manipulation

This project covers ES6 data manipulation techniques including array methods, typed arrays, and data structures like Set and Map.

## Tasks

### 0. Basic list of objects
Function that returns an array of student objects with id, firstName, and location attributes.

### 1. More mapping
Function that returns an array of student ids using the map method.

### 2. Filter
Function that filters students by location using the filter method.

### 3. Reduce
Function that calculates the sum of all student ids using the reduce method.

### 4. Combine
Function that updates student grades by city using filter and map methods combined.

### 5. Typed Arrays
Function that creates an Int8 typed array using ArrayBuffer and DataView.

### 6. Set data structure
Function that creates a Set from an array.

### 7. More set data structure
Function that checks if all array elements exist in a Set.

### 8. Clean set
Function that filters and processes Set values based on a starting string.

### 9. Map data structure
Function that returns a Map of groceries with their quantities.

### 10. More map data structure
Function that updates Map entries with quantity 1 to quantity 100.
