# ES6 Promises

This project covers ES6 Promises and asynchronous JavaScript including Promise creation, handling, and error management.

## Tasks

### 0. Keep every promise you make and only make promises you can keep
Function that returns a Promise.

### 1. Don't make a promise...if you know you can't keep it
Function that returns a promise that resolves or rejects based on a boolean parameter.

### 2. Catch me if you can!
Function that handles Promise responses using then, catch, and finally methods.

### 3. Handle multiple successful promises
Function that uses Promise.all to resolve multiple promises collectively and log the results.

### 4. Simple promise
Function that returns a resolved promise with user data.

### 5. Reject the promises
Function that returns a rejected promise with an error message.

### 6. Handle multiple promises
Function that uses Promise.allSettled to handle multiple promises and return their status and values.

### 7. Load balancer
Function that uses Promise.race to return the value from the first resolved promise.

### 8. Throw an error
Function that throws an error when dividing by zero, otherwise returns the division result.

### 9. Throw error / try catch
Function that uses try-catch-finally to execute a function safely and return results in a queue array.
